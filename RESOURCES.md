## Official Microsoft Learning Resources

### Exam Information
- **Exam Page:** [AB-100: Agentic AI Business Solutions Architect](https://learn.microsoft.com/en-us/credentials/certifications/exams/ab-100/)
- **Skills Measured:** Download the detailed exam objectives PDF
- **Exam Format:** 40-60 questions, 120 minutes, passing score 700/1000
- **Prerequisites:** Recommended experience with Microsoft 365, Azure, and AI solutions

## Resource Study Schedule

### Week-by-Week Resource Focus

**Week 1-2: Domain 1 (Plan)**
- [ ] Complete 2-3 Microsoft Learn modules on data quality and preparation
- [ ] Read Cloud Adoption Framework for AI strategy section
- [ ] Watch 3-5 Azure Friday episodes on AI planning
- [ ] Explore Microsoft Graph and Dataverse documentation

**Week 3-4: Domain 2 (Design)**
- [ ] Complete Copilot Studio learning path
- [ ] Work through M365 Copilot extensibility modules
- [ ] Build sample declarative agent following GitHub examples
- [ ] Explore Azure AI Foundry and Prompt Flow documentation

**Week 5-7: Domain 3 (Deploy)**
- [ ] Complete monitoring and DevOps learning paths
- [ ] Set up Application Insights for a sample project
- [ ] Work through Power Platform ALM documentation
- [ ] Study security and compliance resources

**Week 8: Review and Integration**
- [ ] Revisit bookmarked documentation for weak areas
- [ ] Watch exam prep videos if available
- [ ] Review community discussions on exam topics
- [ ] Take final practice assessments
